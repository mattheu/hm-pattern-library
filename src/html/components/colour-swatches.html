Usage Instructions

HTML Classes. You can use class names to set an element background color.

Background-Primary /* Red */
Background-Accent /* Blue */
Background-WarmGrey
Background-LightGrey
Background-DarkGrey
Background-Beige
Background-MediumGrey
Background-Brown

SASS Variables. You can also use variables in SASS.

$color-primary // Red
$color-accent // Blue
$color-warm-grey
$color-light-grey
$color-dark-grey
$color-beige
$color-medium-grey
$color-brown
