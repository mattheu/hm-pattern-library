Usage Instructions

There are a several different logos variations available for you to use, in 3 different colors.

The logos are scalable SVGs. You can resize these as you need, but be sure to maintain the correct aspect ratio! We've also included some margins around the logos, which you can override as required, but make sure you give it enough space to breath!

Usage You can use the classes documented below directly in your HTML, or you can extend them in your SCSS.

Logo variations. You can use the following class names to style an element using one of the 4 logo variations.HMLogo, HMLogo-Alt, HMLogo-Small and HMLogo-Tiny

Colour variations. Logos are available in black (default), red and white. Each logo variation is available in each color variation, just use the class name in addition to the logo variation class. HMLogo-Red and HMLogo-White. Black does not require a colour variation CSS class.

Examples.

<span class="HMLogo">Human Made</span><!-- Standard logo -->
<span class="HMLogo-Alt">Human Made</span><!-- Alternate logo -->
<span class="HMLogo-Small">Human Made</span><!-- Small logo -->
<span class="HMLogo-Tiny">Human Made</span><!-- Tiny logo -->
<span class="HMLogo">Human Made</span><!-- Black, standard logo -->
<span class="HMLogo HMLogo-Red">Human Made</span><!-- Red, standard logo -->
<span class="HMLogo HMLogo-White">Human Made</span><!-- White, standard logo -->
