Grid

Juniper includes a a very basic 12 column grid system. There are a few utility classes and helper mixins available for you to use.

Utility Classes

.util-GridItem-Row Grid container element class. Required. Offsets margins and clears floats.

.util-GridItem-$i Where $i is the number of columns the grid item should span.

Mixins

grid-row(); Grid item container element. Required. Offsets margins and clears floats.

grid-item( $columns ); Create a fluid grid item that spans the number of columns passed in a 12 column grid. (Optional, pass total columns as 2nd parameter).

Further Information

Responsive grids The utility classes simply drop to a single column on smaller screens. If using the mixins, you can fine tune your own layout yourself. See the style guide css as a reference.

Nested grids You can easily do nested grids using the SASS mixin. Simply pass the current column width (in columns) as the second param. For example, if you have a grid item that spans 6 columns, and you wish to have a child element that spans 1 column of the original 12 column grid, you can use grid-item( 1, 6 );

Gutter width The size of the gutter is controlled using the $gutter-width variable.
