Grid

Juniper includes a a very basic grid and a couple of helper mixins. It is a fluid grid, and each item is floated and uses padding for gutters. The size of the gutter is controlled using the $gutter-width variable

Utility Classes

.util-GridItem-$i Where $i is the number of columns the grid item should span.

.util-GridItem-Row Grid container class. Offsets margins and clears floats.

Mixins

grid-item( $columns, $total_columns ); which will create a floated grid item with a fluid width, and internal padding using the value of $gutter-width.defaults to total 12 columns).

grid-row(); Offsets margins and clears floats.
