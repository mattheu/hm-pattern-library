Usage Instructions

Mark up code with <code> tags for inline code snippets, or <pre> and <code> for longer blocks of code.

For code syntax highlighting we are using Prism.js. Mark up your code block with the class language-$lang. Prisum supports php, js, html, css, scss and many others. Refer to the Prism.JS documentation for a full list of available languages.

Examples

<pre><code class="language-html"><!-- HTML CODE HERE --></code></pre>
<pre><code class="language-js"><!-- JS CODE HERE --></code></pre>
<pre><code class="language-php"><!-- PHP CODE HERE --></code></pre>
